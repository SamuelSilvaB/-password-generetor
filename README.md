<h1 align="center"> Purpose of the project </h1>


<h3> generate random passwords, with possibility to choose password settings like;</h3>  <br> <br>

<h3>
<ul>
* Length of the password (1 - 30)
* Uppercase  (A - Z)
* Numbers (0 - 9)
* Symbol (!-$^+)
* Include Spaces
* Delete Duplicates
</ul>
</h3>
